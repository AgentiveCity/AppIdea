# Buy Now, Pay Later (BNPL) Platform

## Overview
A BNPL platform for online merchants allowing customers to split purchases into interest-free installments. Competes with Klarna, Afterpay, Affirm.

## Key Features
1. **Instant Approval**: Credit decision in <1 second
2. **4 Installments**: Split into 4 payments, no interest
3. **Merchant Integration**: Shopify plugin, API for custom stores
4. **Consumer App**: Manage all BNPL purchases
5. **Credit Building**: Report payments to credit bureaus
6. **Fraud Prevention**: AI-powered risk assessment

## Revenue Model
- **Merchant Fee**: 3-6% per transaction (merchants pay, not consumers)
- **Late Fees**: $7-10 for missed payments

**Path to $100K MRR**: $2M/month GMV @ 5% take rate = $100K
